# Projeto Mega-Sena

O "Projeto Mega-Sena" é um aplicativo Android feito em Java onde o usuário pode realizar um sorteio de 6 números aleatórios em ordem crescente.

# Funcionamento

O aplicativo utiliza a função `Random()` para gerar números aleatórios sem repetição até 60 que são colocados dentro de uma lista chamada numeros com 6 posições (de 0 á 5), que, por sua vez, são exibidos na tela do app por meio do método `setText()`.
